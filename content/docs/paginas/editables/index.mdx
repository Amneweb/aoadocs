## Definición

Las páginas editables consisten en un conjunto de elementos (o cajas de información) dispuestos uno a continuación del otro de manera de generar un resultado visual coherente con contenido adecuado a lo que se quiere mostrar.

## Ejemplo

En el video a continuación se muestran algunos de los elementos de la página Trámites.

- **Primer parte:** Datos básicos, similares a los datos de <Link href="/docs/colecciones/metadata">`Metadata`</Link> de las páginas pre armadas

- **Segunda parte:** Elementos varios que luego de guardados, dan lugar a la página completa. A medida que se van desplegando los elementos, se muestra una imagen de cómo se ve el contenido en la página real. Algunas de las cajas de contenido que se muestran de ejemplo son:

| Elemento            | Función en la página                                                      |
| ------------------- | ------------------------------------------------------------------------- |
| `conjunto de items` | Listado de requisitos para competir                                       |
| `destacado`         | Cuadro negro indicando que no se puede pasar de timoneles a principiantes |
| `bloque de texto`   | Pasos a seguir para pedir un duplicado                                    |
| `caja de imagen`    | Información sobre cómo acceder al sector de selectivos                    |
