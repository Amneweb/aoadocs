## Datos básicos

Listado de datos básicos a ingresar para `crear` un evento. Los datos que se mencionan en la sección siguiente "DATOS SECUNDARIOS" se pueden ingresar después, cuando se tengan documentos, o resultados, o seleccionados, etc. Pero para crear un evento y que salga publicado en la lista de campeonatos, sólo es necesario completar los campos que se mencionan en la tabla a continuación (y algunos de los campos no son obligatorios)

| Nombre del campo       | Recomendaciones                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               | Tipo de dato                              | Obligatorio                                                                       |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------- | --------------------------------------------------------------------------------- |
| `nombre_campeonato`    | El nombre del evento, tratando de que sea único, incluso si el evento se organiza todos los años (conviene agregar el año)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | Texto corto                               | <span className="bg-yellow-100 px-1 border-1 border-yellow-400 rounded">SI</span> |
| `anio_campeonato`      | Puede dejarse vacío, se autocompleta con el año en curso                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      | Número                                    | NO                                                                                |
| `inicio_campeonato`    | Conviene usar el calendario para no equivocarse. Tener cuidado si el formato de fechas es en inglés (MM/DD/AAAA)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              | Fecha                                     | <span className="bg-yellow-100 px-1 border-1 border-yellow-400 rounded">SI</span> |
| `fin_campeonato`       | Idem                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          | Fecha                                     | <span className="bg-yellow-100 px-1 border-1 border-yellow-400 rounded">SI</span> |
| `tipo_campeonato`      | Se debe elegir entre las opciones <br />`ioda` : Mundial, Sudamericano, etc<br />`internacional_local` : Abierto a extranjeros, organizado en Argentina<br />`local` : evento local, sin participación de extranjeros<br />`internacional_exterior` : evento internacional (afuera o en Argentina) pero NO IODA<br />`nacional` : campeonato Argentino                                                                                                                                                                                                                                                                                                                        | Restringido                               | <span className="bg-yellow-100 px-1 border-1 border-yellow-400 rounded">SI</span> |
| `sitio_web_campeonato` | Ingresar la url completa, incluyendo el http://                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               | Texto corto                               | NO                                                                                |
| `logo_campeonato`      | Subir imagen cuadrada (ideal 1080 x 1080 px)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  | .jpg, .jpeg, .png, .wepb. NO se puede svg | NO                                                                                |
| `tipo_ioda`            | Este campo aparece si se eligió IODA en el `tipo_campeonato`. Se debe elegir entre las opciones, que son todos los tipos de evento IODA: `norteamericano`, `mundial`, `sudamericano`, etc                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     | Restringido                               | <span className="bg-yellow-100 px-1 border-1 border-yellow-400 rounded">SI</span> |
| `muestra_pais`         | Si en `tipo_campeonato` se eligió IODA o internacional_exterior, se debe poner el switch en TRUE, para poder seleccionar el país sede                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         | Verdadero / Falso                         | NO                                                                                |
| `pais_campeonato`      | Si `muestra_pais` está encendido (TRUE) se debe seleccionar del menú desplegable. Tener en cuenta que los nombres de los países se muestran en inglés (en la página se ven en español)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        | Restringido                               | NO                                                                                |
| `ciudad_campeonato`    | Si `muestra_pais` está encendido (TRUE) se verá esta casilla para ingresar la ciudad. No hay opciones, se debe ingresar el nombre a mano                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      | Texto corto                               | NO                                                                                |
| `clubes`               | Si el `tipo_campeonato` es IODA o internacional_exterior, esta casilla queda sin completar. Si el evento es local o internacional_local, entonces de debe ingresar el club, haciendo click en la flecha de la derecha y eligiendo la opción que corresponda. Hay alrededor de 20 clubes ya ingresados, pero si el que se desea ingresar no aparece, se lo puede crear en el momento haciendo click en '+ crear relación' o dejar este campo vacío, completar el resto de los datos, guardar el campeonato y <Link href="/docs/colecciones/clubes.mdx">`crear el club`</Link> desde el formulario correspondiente. Una vez creado el club, se puede volver y editar este valor | Restringido                               | NO                                                                                |
| `niveles_campeonato`   | Elegir entre `timoneles`, `principiantes`, `escuelita`, `pre-principiantes` y `juntos` (para cuando corren todos juntos, sin considerar la categoría, por ejemplo en los GPL)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | Restringido                               | <span className="bg-yellow-100 px-1 border-1 border-yellow-400 rounded">SI</span> |
| `modalidad_campeonato` | Elegir entre `flota`, `por equipos` o `ambos`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | Restringido                               | <span className="bg-yellow-100 px-1 border-1 border-yellow-400 rounded">SI</span> |
| `slug_campeonato`      | Poner algo que identifique al evento sin tener que pensar mucho, por ejemplo "sba_2026" o "gran_prix_yco_2026". Esta palabra se usa después para identificar al evento desde otras colecciones de datos. En caso de olvido, el sistema la autocompleta, pero conviene cambiarla por una palabra representativa.                                                                                                                                                                                                                                                                                                                                                               | Texto corto, sin espacios                 | <span className="bg-yellow-100 px-1 border-1 border-yellow-400 rounded">SI</span> |

## Datos Secundarios

Se puede crear un evento sin completar estos datos, pero una vez que vayamos teniendo la información conviene agregarla para que el campeonato quede lo más completo posible.

| Nombre del campo             | Recomendaciones                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | Tipo de dato      | Obligatorio |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------- | ----------- |
| `hay_que_seleccionar`        | Se enciende (TRUE) cuando es un campeonato IODA o cualquier otro evento para el que haya que seleccionara para participar. La mayoría de los campeonatos de nuestro calendario no requieren de esta condición                                                                                                                                                                                                                                                                                      | Verdadero / False | NO          |
| `seleccionados`              | Si en `hay_que_seleccionar` se eligió TRUE, acá irán apareciendo los competidores seleccionados, una vez que se los ingrese desde el formulario de <Link href="/docs/colecciones/seleccionados">`Seleccionados`</Link> (no conviene ingresar estos datos desde acá)                                                                                                                                                                                                                                | Relación          | NO          |
| `documentos`                 | Desde acá se eligen los documentos ya cargados desde el formulario <Link href="/docs/colecciones/documentos">`Documentos`</Link>. Para eso hacer click en la flecha y elegir el documento en base al **slug**. Se puede elegir más de un documento.                                                                                                                                                                                                                                                | Relación          | NO          |
| `pertenece_a_series`         | Si el campeonato pertenece a una serie o ranking, seleccionarla desde el menú desplegable. La serie debe haber sido creada anteriormente desde el formulario de <Link href="/docs/colecciones/series">`Series`</Link>. Si la serie se crea después del campeonato, se puede elegir el campeonato desde la Serie y este campo quedará automáticamente completado                                                                                                                                    | Relación          | NO          |
| `pertenece_a_selectivos`     | Valen las mismas consideraciones que para el item anterior. En este caso el selectivo se debe haber creado desde el formulario de <Link href="/docs/colecciones/series#selectivos">`Selectivos`</Link>.                                                                                                                                                                                                                                                                                            | Relación          | NO          |
| `lleva_resultados`           | Si no tenemos resultados, conviene dejar este switch en FALSE. Si por el contrario ya tenemos resultados y los cargamos desde el formulario correspondiente de <Link href="/docs/colecciones/resultados">`Resultados`</Link>, lo ponemos en TRUE y nos va a aparecer un campo para elegir el resultado correspondiente.                                                                                                                                                                            | Verdadero/Falso   | NO          |
| `resultados`                 | Aparece sólo si `lleva_resultados` está encendido. Desde acá se "agarran" los resultados del campeonato, que ya se habían ingresado en Resultados, como se explicó en el renglón anterior. Se puede ingresar uno o más resultados.                                                                                                                                                                                                                                                                 | Relación          | NO          |
| `lleva_titulos`              | Si se trata de un evento cuyos títulos aparecerán en el cuadro de honor (por ejemplo un Argentino), se puede llevar este switch a TRUE. Sin embargo, el no hacerlo no impide la asignación de títulos correspondiente. Este sector del formulario conviene dejarlo como está y generar los títulos desde el formulario de <Link href="docs/colecciones/titulos">`Títulos`</Link>. Una vez hecho esto, los campos de `grupo_de_titulos` y `titulos_otorgados` quedarán automáticamente completados. | Relación          | NO          |
| `entrenadores_seleccionados` | Vale lo mismo que para `Seleccionados`, conviene elegirlos desde su propio formulario, y este campo quedará automáticamente completado                                                                                                                                                                                                                                                                                                                                                             | Relación          | NO          |
| `fechas_inscripciones`       | Si se trata de un evento relacionado con un selectivo, con mediciones o con pagos obligatorios con vencimiento, conviene ingresar los datos de las fechas relevantes en el formulario `Fechas` y luego relacionarlas con el campeonato eligiéndolas desde el menú desplegable.                                                                                                                                                                                                                     | Relación          | NO          |
| `noticias_relacionadas`      | Si se publicó una noticia relacionada con el campeonato, se la puede "agarrar" con el menú desplegable para que en la página del campeonato aparezca la mención a la noticia.                                                                                                                                                                                                                                                                                                                      | Relación          | NO          |
