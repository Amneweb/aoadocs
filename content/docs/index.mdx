## Conceptos básicos

Toda la información se guarda en una base de datos. Los datos a guardar pueden ser:

- 🗂️ **Datos básicos o primarios:** Son datos que corresponden a hechos reales, por ejemplo valores de cuotas, fechas de campeonatos, nombres de competidores, etc. Estos datos se pueden filtrar y ordenar para mostrar la información de diferentes maneras.
- 📓 **Datos para contenido:** Textos e imágenes.
- 🏗️ **Datos para el layout:** Se guarda información sobre qué datos de contenido vamos a mostrar, en qué orden y, en algunos casos, qué colores se van a usar.
